# 💼 PHP Job Portal Application

A web-based Job Portal Application developed using **PHP, MySQL, HTML, and  CSS**. This application allows job seekers to register, log in, browse available jobs, and manage their profiles. It also provides employers with the ability to post and manage job listings through a simple and user-friendly interface.

---

## 🌐 Live Demo

⚠️ This project is built using **PHP and MySQL**, so it cannot be hosted on GitHub Pages.

Run the project locally using **XAMPP**:

```text
http://localhost/Ganesh/
```

## 🚀 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/ganeshdesai-2006/php-job-portal-application.git
```

### 2. Copy the Project

Move the project folder into:

```text
C:\xampp\htdocs\
```

### 3. Start XAMPP

Start the following services:

- Apache
- MySQL

### 4. Import Database

- Open **http://localhost/phpmyadmin**
- Create the required database.
- Import the provided SQL file (if available).

### 5. Run the Project

Open your browser and visit:

```text
http://localhost/Ganesh/
```
